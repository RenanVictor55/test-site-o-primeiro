sites e aplicativos de Aprender programação! 🚀

  1. 1. Mimo

    O Mimo ensina programação com aulas interativas e desafios diários.

    👍 Prós: Fácil de usar, várias linguagens.

    👎 Contras: Algumas aulas são pagas.

    Baixar Mimo

  2. 2. SoloLearn

    O SoloLearn tem cursos grátis de Python, JavaScript e mais.

    👍 Prós: Comunidade ativa, quizzes legais.

    👎 Contras: Versão grátis tem limites.

    Baixar SoloLearn

  3. 3. Programming Hero

    Esse app ensina programação com jogos e desafios.

    👍 Prós: Divertido e interativo.

    👎 Contras: Algumas aulas são pagas.

    Baixar Programming Hero

  4. 4. Grasshopper

    App do Google que ensina JavaScript para iniciantes.

    👍 Prós: 100% grátis e fácil de usar (para ios e android)

    👎 Contras: Só tem JavaScript.

    Baixar Grasshopper

  5. 5. Codecademy

    Uma plataforma cheia de cursos de programação.

    👍 Prós: Cursos bem explicados.

    👎 Contras: A versão grátis tem poucas opções.

    Baixar Codecademy
